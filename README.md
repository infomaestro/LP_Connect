Important note....

Certificates are not included 

<b>To make the app work you need to get and upload the certificates named app.crt and app.key</b>

The app is for testing purposes only and is in no way associated with Leapmotor. Use is for testing purposes only, and no liability is assumed.
If you're unsure, don't test the app.

WINDOWS VERSION
Remember to add  libcrypto-1_1.dll in Windows release folder /Win32 or /Win64
Available at
- [Overbyte OpenSSL 1.1.1w Win32](https://wiki.overbyte.eu/arch/openssl-1.1.1w-win32.zip)


ANDROID VERSION
Remember to add libcrtpto.so and libssl.so in for arm64-v8a and armeabi-v7a
Prebuilt libs are available at
- [OpenSSL-for-Android-Prebuilt](https://github.com/PurpleI2P/OpenSSL-for-Android-Prebuilt)
